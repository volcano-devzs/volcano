### ❌ Problem: Script Throws Errors

**Symptoms:** Scripts execute but produce Lua errors in the console.

### ❌ Problem: Key Verification Fails

- Check the discord for server status announcements
- If the **volcano executor key** still fails, re-download a fresh build from GitHub
